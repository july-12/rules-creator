关于 Rules Creator 的常见疑问解答

Rules Creator 是一款专为 Coohom 平台设计的 Chrome 浏览器扩展程序，帮助用户快速创建和管理页面元素规则配置。通过直观的可视化界面，用户可以轻松选择页面元素并设置相应的规则参数。
您可以从 Chrome 网上应用店搜索"Rules Creator"并点击安装。安装完成后，在 coohom.com 网站上点击浏览器工具栏中的扩展图标即可开始使用。选择页面元素，配置规则参数，然后保存即可。
Rules Creator 仅处理必要的功能数据，包括当前页面的 URL 路径、您主动选择的页面元素信息以及创建的规则配置。所有数据仅用于提供扩展的核心功能，不会用于广告投放或跨站追踪。详情请参阅我们的隐私政策。
目前 Rules Creator 主要支持 Chrome 浏览器。同时，基于 Chromium 内核的浏览器（如 Microsoft Edge、Brave 等）也可能兼容使用。我们计划在未来扩展对更多浏览器的支持。
您的规则配置会通过安全的方式保存到 Coohom 服务器，同时也会在本地浏览器存储中保留缓存副本，确保在网络不稳定时也能正常使用。您随时可以在扩展中查看、编辑或删除已创建的规则。
如果您在使用过程中遇到任何问题，可以通过邮件 seventanyb@gmail.com 联系我们的技术支持团队。我们会尽快回复您的问题并提供帮助。
