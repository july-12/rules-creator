Rules Creator Privacy Policy

This Privacy Policy explains how the Rules Creator Chrome extension handles information when users configure page element rules on coohom.com and its subdomains.

Effective date: March 15, 2026
Rules Creator only works on coohom.com and its subdomains. It is designed solely to help users select page elements and manage page-specific rule configurations.

1. Information We Process

When you use the extension, the following information may be processed:

  • The URL and pathname of the active Coohom page, to identify which page's rules should be loaded or saved.
  • Information about page elements you actively select, such as tag name, id, class, attributes, visible text snippets, and generated selectors.
  • Rule configuration data that you create, edit, or delete, including selectors, rule types, and rule parameters.
  • Local cache data required to keep the extension functioning reliably.

2. How We Use Information

We use this information only to provide the extension's core functionality, including:

  • Displaying, creating, editing, and deleting rules for the current page.
  • Loading rule configuration based on the current page pathname.
  • Saving updated rule configuration to the same-origin backend API.
  • Using local cache as a fallback when necessary for stability and user experience.

3. Data Storage and Transmission

  • The extension uses Chrome local storage to store necessary cache data.
  • The extension communicates with same-origin Coohom API endpoints to load and save page rule configuration.
  • The extension does not use collected information for advertising, profiling, or cross-site tracking.

4. Permissions

The extension requests only the permissions required to deliver its single purpose:

  • activeTab: to access the currently active page when the user explicitly opens the extension or uses its shortcut.
  • tabs: to obtain the active tab URL and tab id, and to send actions to the current page.
  • storage: to store local cache data needed by the extension.
  • Host permissions: limited to coohom.com and its subdomains, where the extension runs and interacts with same-origin APIs.

5. Data Sharing

We do not sell, rent, or share your data with unrelated third parties. Information is processed only as necessary to support the extension's functionality within Coohom services.

6. What We Do Not Do

  • We do not run on websites outside coohom.com and its subdomains.
  • We do not collect browsing history for advertising or analytics purposes.
  • We do not use extension data for marketing or unrelated profiling.
  • We do not access unrelated website content beyond what is required for the selected page and the extension feature.

7. Your Choices

You may stop using the extension at any time. You may also delete configured rules, clear browser extension storage, or uninstall the extension.
